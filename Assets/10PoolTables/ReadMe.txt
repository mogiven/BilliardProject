INDOOR SPORTS SERIES
====================

This is a fantastic indoor sports venue room from Super Icon Ltd.

Lighting is all set-up and optimised for the free version of Unity. Scene includes the baked lightmaps. All furniture objects are Prefabs.
Textures are not atlased to ensure a good and consistent Texel resolution, many include normal maps. Reflection map also included.

The package comes with an example demo scene.
